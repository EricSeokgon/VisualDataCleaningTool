# 📊 Visual Data Cleaning Tool

CSV 데이터를 업로드하고 결측치와 이상치를 시각적으로 감지하여 클리닝하는 웹 애플리케이션

## ✨ 주요 기능

### 1. 📁 CSV 파일 업로드
- 드래그 앤 드롭 지원
- 클릭하여 파일 선택
- 자동 파싱 및 데이터 분석

### 2. 🔍 자동 감지
- **결측치 감지**: NULL, 빈 값, "NA", "N/A" 등 자동 탐지
- **이상치 감지**: IQR(Interquartile Range) 방법을 사용한 통계적 이상치 탐지
- 실시간 통계 계산 및 시각화

### 3. 🎨 시각적 하이라이트
- 🔴 결측치: 빨간색으로 표시
- 🟠 이상치: 주황색으로 표시
- 🟢 변경된 값: 초록색으로 표시

### 4. 🛠️ 데이터 클리닝 옵션

#### 결측치 처리
- **평균값으로 채우기**: 숫자형 열의 평균값으로 결측치 대체
- **중앙값으로 채우기**: 숫자형 열의 중앙값으로 결측치 대체
- **최빈값으로 채우기**: 가장 자주 나타나는 값으로 결측치 대체
- **행 삭제**: 결측치가 있는 행 전체 삭제

#### 이상치 처리
- **경계값으로 제한**: 이상치를 IQR 경계값으로 제한
- **행 삭제**: 이상치가 있는 행 전체 삭제

### 5. 👁️ Before/After 비교
- 원본 데이터와 클리닝된 데이터를 나란히 비교
- 변경된 셀 자동 하이라이트
- 토글 버튼으로 비교 뷰 열기/닫기

### 6. 📊 데이터 품질 대시보드
- **품질 점수**: 0-100점 척도로 데이터 품질 평가
- **메트릭 카드**:
  - 총 행 수 / 열 수
  - 결측치 개수 및 비율
  - 이상치 개수 및 비율
- 진행률 바로 시각화

### 7. 💾 데이터 내보내기
- 클리닝된 데이터를 CSV 파일로 다운로드
- 타임스탬프가 포함된 파일명 자동 생성

## 🚀 사용 방법

### 1. 애플리케이션 실행
```bash
# 프로젝트 디렉토리로 이동
cd data-cleaning-tool

# index.html 파일을 브라우저에서 열기
# 방법 1: 파일 탐색기에서 index.html 더블클릭
# 방법 2: 브라우저에서 파일 열기 (Ctrl+O)
```

### 2. CSV 파일 업로드
- 업로드 영역을 클릭하거나 CSV 파일을 드래그 앤 드롭
- 샘플 데이터: `sample_data.csv` 파일 사용 가능

### 3. 데이터 분석 확인
- 자동으로 생성된 품질 점수 확인
- 결측치 및 이상치 통계 확인
- 데이터 테이블에서 하이라이트된 셀 확인

### 4. 데이터 클리닝
- 결측치 처리 방법 선택 후 "적용" 버튼 클릭
- 이상치 처리 방법 선택 후 "적용" 버튼 클릭
- 실시간으로 변경사항 확인

### 5. Before/After 비교
- "Before/After 비교" 버튼 클릭
- 원본과 클리닝된 데이터 비교

### 6. 데이터 내보내기
- "클리닝된 데이터 다운로드" 버튼 클릭
- 자동으로 CSV 파일 다운로드

## 📊 알고리즘

### 결측치 감지
```javascript
// 다음 값들을 결측치로 간주:
- null
- undefined
- 빈 문자열 ("")
- "NA", "N/A"
- "null", "NaN"
```

### 이상치 감지 (IQR 방법)
```javascript
Q1 = 25번째 백분위수
Q3 = 75번째 백분위수
IQR = Q3 - Q1

하한 = Q1 - 1.5 × IQR
상한 = Q3 + 1.5 × IQR

// 하한보다 작거나 상한보다 큰 값을 이상치로 간주
```

### 품질 점수 계산
```javascript
완전성 = (총 셀 수 - 결측치 수) / 총 셀 수 × 100
이상치 페널티 = 이상치 수 / 총 셀 수 × 100

품질 점수 = 완전성 - 이상치 페널티 (0-100 범위)
```

## 📝 샘플 데이터 정보

`sample_data.csv` 파일에는 30개의 직원 데이터가 포함되어 있습니다:
- **열**: Name, Age, Salary, Department, Experience, Score
- **의도적 결측치**: 여러 행에 빈 값 포함
- **의도적 이상치**: Salary 열에 250000 (극단값) 포함
